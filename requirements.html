CryoTracker runs on Windows 7, 8, & 10

Microsoft .NET Framework 4.5

CryoTracker requires Microsoft .NET Framework 4.5 or higher. Click here to download the installer from Microsoft. Make sure you review the supported operating systems and hardware requirements for .NET Framework 4.5.

Not sure what version of the .NET Framework you have installed?
Go to Control Panel > Programs and Features > Search "Microsoft .NET Framework".

Microsoft Access Runtime 2007

CryoTracker stores your slot items in Microsoft Access databases. Click here to download the installer.
