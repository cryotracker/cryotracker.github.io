CryoTracker is a cryo/freezer box contents tracker for Windows Desktop

CryoTracker is a simple program that allows you to keep track of all your materials

Below are some of the features available in CryoTracker

  • Create boxes that are organized by rack and location
  • Choose between 10 box divisions, from an empty box to a 10x10 configuration
  • Global search all slot contents to find a specific item
  • Combine box slots to mimic your actual workflow for storing items
  • Archive items to keep a log of those that are no longer active
  • Move or copy items from one slot to another
  • Add file attachments for each box as well as individual slot items
